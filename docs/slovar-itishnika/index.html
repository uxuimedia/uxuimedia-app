Доки

Словарь айтишника

HTML

HTML — язык разметки гипертекста (Hypertext Markup Language). Стандартный язык разметки документов, предназначенных для отображения в веб-браузере. Он определяет содержание и структуру веб-контента. К нему часто добавляются такие технологии, как каскадные таблицы стилей (CSS) и языки сценариев, такие как JavaScript.

CSS

CSS — это язык, который используется для стилизации веб-страниц.

  • CSS расшифровывается как каскадные таблицы стилей
  • CSS описывает, как элементы HTML должны отображаться на экране, планшете или других носителях
  • CSS экономит много работы, с его помощью можно управлять оформлением сразу нескольких веб-страниц
  • Внешние таблицы стилей хранятся в файлах с расширением CSS

Bootstrap 5

Bootstrap — это бесплатный фреймворк (набор инструментов) с открытым исходным кодом для создания адаптивных веб-сайтов и веб-приложений, включающий готовые компоненты из HTML, CSS и JavaScript для типографики, форм, кнопок, навигации и других элементов интерфейса, что ускоряет процесс разработки. Он позволяет быстро верстать сайты, которые корректно отображаются на любых устройствах, от смартфонов до настольных компьютеров.

Javascript

JavaScript — это высокоуровневый, мультипарадигменный язык программирования, используемый для создания интерактивных веб-страниц и веб-приложений, добавляющий динамику и функциональность сайтам, таким как анимации, всплывающие формы и игры. Он является кросс-платформенным, легковесным и широко применяется как на стороне клиента (в браузерах), так и на стороне сервера (с использованием Node.js) для разработки различных проектов, включая мобильные приложения и серверные приложения.

PHP

PHP — это популярный язык программирования общего назначения с открытым исходным кодом, созданный специально для веб-разработки, который позволяет создавать динамические веб-сайты, веб-приложения и управлять ими. Он интегрируется в HTML, обрабатывает запросы на сервере, работает с базами данных и пользовательским вводом, что делает его мощным инструментом для веб-разработчиков.

MySQL

MySQL — это популярная система управления реляционными базами данных (СУБД), которая используется для хранения, организации и управления большими объемами данных в виде связанных таблиц. Эта система использует язык запросов SQL для выполнения операций с данными и работает по принципу "клиент-сервер". MySQL доступна в бесплатной версии (Community Edition) и платной корпоративной (Enterprise Edition), что делает ее универсальным инструментом для веб-сайтов, приложений и корпоративных систем.

Wordpress

WordPress — это бесплатная система управления содержимым сайта (CMS) с открытым исходным кодом, предназначенная для создания и администрирования веб-сайтов, от блогов до новостных порталов и интернет-магазинов. Она популярна благодаря простоте использования, интуитивно понятному интерфейсу и возможности расширения функциональности с помощью готовых тем и плагинов без необходимости знания программирования.

Git

Git — это бесплатная распределенная система управления версиями с открытым исходным кодом, которая позволяет отслеживать изменения в файлах, сохранять историю версий проекта и возвращаться к предыдущим состояниям. Будучи распределенной, каждый разработчик имеет полную копию истории проекта локально, что упрощает работу как в автономном, так и в удаленном режиме. Git широко используется в разработке программного обеспечения, включая такие крупные проекты, как ядро Linux, Android и Swift.

Linux

Linux — это семейство свободных операционных систем с открытым исходным кодом, созданных на основе одноименного ядра, разработанного Линусом Торвальдсом. Linux используется на серверах, смартфонах, компьютерах и других устройствах, предоставляя пользователям гибкость, высокий уровень безопасности и возможность выбора из множества специализированных дистрибутивов — собранных вместе ядра и дополнительных программ.

VSCode

Visual Studio Code (VS Code) — это бесплатный, кросс-платформенный редактор кода от Microsoft, который поддерживает множество языков программирования и имеет богатую экосистему расширений. Он предоставляет такие возможности, как подсветка синтаксиса, автодополнение кода (IntelliSense), интеграция с Git и встроенный отладчик. VS Code работает на Windows, macOS и Linux, и его можно использовать для разработки веб-приложений, десктопных и других проектов.

Sublime Text

Sublime Text – это кроссплатформенный текстовый редактор кода с минималистичным интерфейсом и широкими возможностями настройки, предназначенный для программистов и веб-разработчиков. Он отличается высокой производительностью, поддержкой множества языков программирования, мощным поиском, автодополнением кода и возможностью расширять функциональность с помощью плагинов на Python.

Notepad++

Notepad++ – это бесплатный текстовый редактор с открытым исходным кодом для Windows, предназначенный для редактирования как обычного текста, так и программного кода, предлагающий расширенные функции по сравнению с простым Блокнотом, такие как подсветка синтаксиса для множества языков программирования, сворачивание кода и работа с несколькими файлами во вкладках. Он пользуется популярностью среди разработчиков благодаря своей функциональности и возможности расширения с помощью плагинов.

Photoshop

Photoshop — это мощный и широко используемый растровый графический редактор, разработанный компанией Adobe, предназначенный для работы с изображениями, их редактирования, создания цифровой графики, ретуши фотографий и даже анимации. Он позволяет пользователям создавать и обрабатывать изображения, состоящие из пикселей, предлагая широкий спектр инструментов для дизайна, художественного творчества и профессиональной работы с фото.

Illustrator

Illustrator (Adobe Illustrator) — это отраслевой стандарт программного обеспечения для создания и редактирования векторной графики, которая состоит из математических кривых, а не из пикселей, что позволяет масштабировать изображения без потери качества. Дизайнеры используют Illustrator для создания логотипов, иконок, иллюстраций, макетов и других графических материалов для печати и цифровых носителей.

Google

Google LLC — американская многонациональная корпорация и технологическая компания, специализирующаяся на интернет-рекламе, поисковых технологиях, облачных вычислениях, программном обеспечении, квантовых вычислениях, электронной коммерции, потребительской электронике и искусственном интеллекте (ИИ).

Ее называют «самой мощной компанией в мире», и она является одним из самых ценных брендов в мире благодаря своему доминированию на рынке, сбору данных и технологическим преимуществам в области ИИ. Материнская компания Google, Alphabet Inc., входит в пятерку крупнейших технологических компаний наряду с Amazon, Apple, Meta и Microsoft.

Go

Синонимы: Го, Golang

Go или Golang — статически типизированный компилируемый язык программирования высокого уровня, разработанный в Google Робертом Гриземером, Робом Пайком и Кеном Томпсоном. Синтаксически он похож на C, но также имеет защиту памяти, сборку мусора, структурную типизацию и параллелизм в стиле CSP. Его часто называют Golang из-за его прежнего доменного имени golang.org, но его правильное название — Go.

Существует две основные реализации:

  • Самостоятельный инструментарий компилятора Google «gc», ориентированный на множество операционных систем, и WebAssembly.
  • gofrontend, фронтенд для других компиляторов, с библиотекой libgo. С GCC это сочетание — gccgo; с LLVM — gollvm.

Сторонний компилятор GopherJS, компилирующий Go в JavaScript для фронтенд-веб-разработки.
